# Setup.exe (Windows)
:::warning
CLI reference is currently only available for Windows. Arguments may be slightly different for other platforms.
Please check the help text yourself for the most accurate information.
:::
```
Velopack Setup (0.0.556) installs applications.
https://github.com/velopack/velopack

Usage: setup.exe [OPTIONS]

Options:
  -s, --silent           Hides all dialogs and answers 'yes' to all prompts
  -v, --verbose          Print debug messages to console
  -l, --log <FILE>       Enable file logging and set location
  -t, --installto <DIR>  Installation directory to install the application
  -h, --help             Print help


```
